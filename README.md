# 🛒 Amazon E-Commerce Sales Dashboard

## 📌 Project Overview

This project analyzes Amazon E-Commerce sales data using Python to uncover meaningful business insights. The dataset was cleaned, processed, and visualized using **Pandas**, **NumPy**, and **Matplotlib**. The dashboard provides a comprehensive overview of sales performance, product categories, order status, and geographical sales distribution.

---

## 🎯 Objectives

- Analyze Amazon E-Commerce sales data.
- Identify the top-performing states and cities.
- Understand category-wise sales distribution.
- Analyze order status distribution.
- Visualize business insights through an interactive dashboard.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- VS Code

---

## 📂 Dataset

The dataset contains Amazon E-Commerce sales records with information such as:

- Order ID
- State
- City
- Product Category
- Sales Amount
- Order Status
- Customer Details

---

## 📊 Dashboard Features

- 📍 Top 10 States by Sales
- 🏙️ Top 10 Cities by Sales
- 📦 Category-wise Sales Distribution
- 🚚 Order Status Analysis
- 📈 Sales Performance Visualization

---

## 💡 Key Insights

- Maharashtra recorded the highest overall sales.
- Bengaluru emerged as the top-performing city.
- **Set** is the highest-selling product category.
- Most customer orders were successfully shipped.
- Sales are concentrated in a few major states and cities.

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/komalprajapati39131-stack/Amazon-ECommerce-Sales-Dashboard.git
```

2. Navigate to the project folder

```bash
cd Amazon-ECommerce-Sales-Dashboard
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib
```

4. Run the Python script

```bash
python amazon_dashboard.py
```
