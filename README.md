# 🛒 Amazon E-Commerce Sales Dashboard

## 📌 Project Overview

This project analyzes Amazon E-Commerce sales data using Python. The dataset was cleaned, processed, and analyzed with Pandas, while Matplotlib was used to create a dashboard that provides valuable business insights into sales performance, product categories, order status, and geographical trends.

## 🎯 Objectives

- Analyze Amazon sales data
- Identify top-performing states and cities
- Understand category-wise sales distribution
- Analyze order status
- Visualize business insights through charts

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- VS Code

## 📊 Dashboard Features

- Top 10 States by Sales
- Category-wise Sales Analysis
- Order Status Distribution
- Top Cities by Sales

## 💡 Key Insights

- Maharashtra recorded the highest sales.
- "Set" is the highest-selling product category.
- Most orders were successfully shipped.
- Bengaluru generated the highest city-wise sales.
